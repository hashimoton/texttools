******************
Text Tools
******************

Miscellaneous commands for processing text files.


===============
hook
===============

Continue lines if the following line contains the given pattern.

Usage::
  
  $ hook -h
  Usage: hook [options] FILE [FILE...]

  Options:
      -d=DELIMITER                     Delimiter(--HOOKED--)
      -e=REGEX                         Pattern
      -v                               Invert match
